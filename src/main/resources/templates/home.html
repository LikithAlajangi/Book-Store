Welcome To Archie's Book Store

Archie's Book Store is a web-based application designed to store and manage books efficiently using CRUD (Create, Read, Update, Delete) operations. It allows users to add new books, view book details, update existing entries, and remove books from the catalog. The system ensures seamless inventory management, making it easy for administrators to keep track of available books. Built using modern web technologies like Spring Boot, Thymeleaf, and a database (MySQL), it offers a user-friendly interface for smooth navigation.
